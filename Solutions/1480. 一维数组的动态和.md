# [1480. 一维数组的动态和](https://leetcode-cn.com/problems/running-sum-of-1d-array/)

- 标签：数组
- 难度：简单

## 题目大意

给定一个数组 nums，返回前缀合数组。

## 解题思路

新建数组，递推即可。

## 代码

```Python
class Solution:
    def runningSum(self, nums: List[int]) -> List[int]:
        n = len(nums)
        dp = [0 for _ in range(n)]
        for i in range(n):
            if i == 0:
                dp[i] = nums[i]
            else:
                dp[i] = dp[i-1] + nums[i]
        return dp
```
