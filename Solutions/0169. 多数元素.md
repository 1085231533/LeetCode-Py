## [0169. 多数元素](https://leetcode-cn.com/problems/majority-element/)

- 标签：数组、哈希表
- 难度：简单

## 题目大意

给定一个数组，找到其中相同元素个数最多的元素。

## 解题思路

可以利用哈希表。

遍历一遍数组 nums，用哈希表统计每个元素 num 出现的次数，再遍历一遍哈希表，找出元素个数最多的元素即可。

时间复杂度为：O(N)

空间复杂度为：O(N)

## 代码

```Python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        numDict = dict()
        for num in nums:
            if num in numDict:
                numDict[num] += 1
            else:
                numDict[num] = 1
        max = 0
        max_index = -1
        for num in numDict:
            if numDict[num] > max:
                max = numDict[num]
                max_index = num
        return max_index
```
