## Leetcode 刷题课程第一期

### 课程信息

- 学习周期：14 天，每天平均花费时间 1 小时 - 3 小时不等，根据个人学习接受能力强弱有所浮动。
- 学习形式：理论学习 + 题目刷题
- 人群定位：有 Python 语言编程基础，想要学习算法、数据结构基础知识，想在 LeetCode 刷算法题的学员。
- 先修内容：[Python 编程语言](https://github.com/datawhalechina/learn-python-the-smart-way)
- 难度系数：⭐⭐
